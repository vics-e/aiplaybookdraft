Agent

A system that can work through steps towards a goal, often using tools, memory and rules rather than only answering a single prompt.
ExampleAn agent monitors a shared inbox, drafts replies, checks a policy source and asks for approval before sending.
Watch outDo not let it take high-risk actions without review, permissions and clear limits.
Plain EnglishAssistants respond. Agents act through steps.
